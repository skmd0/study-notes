#+title: create a systemd target.wants directory
#+filetags: :procedure:linux:systemd:

* Problem
  Create a new target.wants directory which groups systemd service unit files.

* Solution
  
1. enter default system's systemd unit directory
   cd /etc/systemd/system/
   
2. create a directory that starts with target filename and ends with .wants
   sudo mkdir my-target.target.wants/
   
3. specify the dependencies by creating the symbolic links to unit files
   sudo ln -s /usr/lib/systemd/system/httpd.service .
