#+title: systemd list of active targets
#+filetags: :procedure:systemd:linux:

* Problem
  Get a list of all the active targets that got loaded.

* Solution
  systemctl list-units --type target

* Additional options

** Output ALL targets (active and inactive)
   systemctl list-units --type target --all
